## Краткая информация

*Данный проект реализует библиотеку функции для работы с квадратами и окружностями*

### Базовый функционал

#### square.py

Имеет две функции:

Функция **area** \(a\)
*Принимает a (double) - длину стороны квадрата и возвращает площадь квадрата в (double)*

Пример:
```python
print(area(10.0))
#console: 100.0
```


Функция **perimetr**\(a\)
*Принимает a (double) - длину стороны квадрата и возвращает периметр квадрата в (double)*
```python
print(perimetr(10.0))
#console: 40.0
```

#### rectangle.py

Имеет две функции:

Функция **area** \(a, b\)
*Принимает a, b (double) - длины сторон прямоугольника и возвращает площадь прямоугольника в (double)*

Пример:
```python
print(area(10.0, 10.0))
#console: 100.0
```


Функция **perimetr**\(a, b\)
*Принимает a, b (double) - длины сторон прямоугольника и возвращает периметр прямоугольника в (double)*
```python
print(perimetr(10.0, 10.0))
#console: 40.0
```

#### circle.py

Имеет две функции:

Функция **area**\(r\)
*Принимает радиус r (double) и возвращает площадь окружности в (double)*

Пример:
```python
print(area(10.0))
#console: 314.159265359
```


Функция **perimetr**\(r\)
*Принимает радиус r (double) и возвращает периметр окружности в (double)*
```python
print(perimetr(10.0))
#console: 62.83118530718
```

#### triangle.py

Имеет две функции:

Функция **area** \(a, h\)
*Принимает a, h (double) - длину стороны и длину высоты треугольника возвращает площадь треугольника в (double)*

Пример:
```python
print(area(10.0, 20.0))
#console: 100.0
```


Функция **perimetr**\(a, b, c\)
*Принимает a, b, c (double) - длины сторон треугольника и возвращает периметр треугольника в (double)*
```python
print(perimetr(1.0, 2.0, 3.0))
#console: 6.0
```
